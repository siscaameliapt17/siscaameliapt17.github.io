Selamat Datang di Sisca Library!

Halo! Saya Sisca Amelia Putri. Saya merupakan mahasiswa Program Studi Perpustakaan dan Sains Informasi di Universitas Sumatera Utara. Sebagai bagian dari perjalanan akademik saya, saya mengerjakan sebuah proyek website ini sebagai bagian dari pemenuhan tugas Mata Kuliah Pengembangan Web. Melalui proyek ini, saya berkesempatan untuk mempelajari dan mengaplikasikan berbagai keterampilan teknis, seperti HTML, CSS, serta teknologi web lainnya, yang mendukung pembelajaran saya dalam mengembangkan aplikasi berbasis web. Website ini juga menjadi sarana untuk mengasah kemampuan saya dalam mendesain dan menyusun platform digital yang bermanfaat.
